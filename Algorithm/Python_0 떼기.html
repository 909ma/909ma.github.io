문제 바로가기

문제 설명:
정수로 이루어진 문자열 n_str이 주어질 때, n_str의 가장 왼쪽에 처음으로 등장하는 0들을 뗀 문자열을 return하도록 solution 함수를 완성해주세요.

문제 풀이:

자료형태를 변경하여 문제를 풀었습니다.

def solution(n_str):
    answer = str(int(n_str))
    return answer


제 코드는 숫자가 아닌 경우 0을 뗄 수 없다는게 문제입니다. 이런 경우 쓸 수 있는 lstrip()이 있습니다.

def solution(n_str):
    return n_str.lstrip('0')

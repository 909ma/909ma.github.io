그대로 출력하기

문제 바로가기

문제 설명:
입력 받은 대로 출력하는 프로그램을 작성하시오.

문제 풀이:

Console.ReadLine()을 통해 한 글자씩 입력을 받자말자 그대로 출력을 하여 입력 버퍼를 다 비울 때까지 무한 반복을 하였습니다.

using System;

namespace MyCompiler {
    class Program {
        public static void Main(string[] args) {
            int inputChar;

            while ((inputChar = Console.Read()) != -1) {
                char character = (char)inputChar;
                Console.Write(character);
            }
        }
    }
}


저는 풀면서 while() 안에 선언과 동시에 조건을 입력하려고 했지만, while()은 선언을 지원하지 않아 외부 선언을 해야함을 알았습니다.
그렇게 고민한 풀이가 위의 풀이였으나 더 좋은 풀이를 발견하여 공유합니다.
풀이 출처

Console.Write(Console.In.ReadToEnd());

Console.In.ReadToEnd()를 하면 입력 버퍼를 한번에 다 받아온다는걸 처음 알았습니다.

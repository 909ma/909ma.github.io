문제 설명:
이 됩니다. 예를 들어 10이 있다면 다음과 같은 과정으로 1이 됩니다.
10 / 2 = 5
(5 - 1) / 2 = 4
4 / 2 = 2
2 / 2 = 1
위와 같이 4번의 나누기 연산으로 1이 되었습니다.
정수들이 담긴 리스트 num_list가 주어질 때, num_list의 모든 원소를 1로 만들기 위해서 필요한 나누기 연산의 횟수를 return하도록 solution 함수를 완성해주세요.

문제 풀이:

문제에서 나온대로 무식하게 풀었습니다.

def solution(num_list):
    answer = 0
    for item in num_list:
        temp = item
        while temp != 1:
            if temp % 2:
                temp = (temp - 1) // 2
                answer += 1
            else:
                temp //= 2
                answer += 1

    return answer


그러나 다른 사람의 풀이를 찾아보니 문제는 2진수로 나타냈을 때의 자릿수를 구하는 문제였기에 아래와 같이 풀이가 가능했습니다.

def solution(num_list):
    return sum(len(bin(i)) - 3 for i in num_list)
